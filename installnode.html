Instalar NodeJS

Ya que creamos al nuevo usuario, podemos continuar con la correcta instalación de NodeJS

En este blog vamos a instalar NodeJS por medio de NVM (Node Version Manager) para lo que vamos a escribir en la consola: (no olvides que debes estar logeado como super usuario con sudo su)

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

Ya que lo instalamos, vamos a verificar la version de NVM:

nvm -v

Para ver las versiones de node que ahora tenemos disponibles, corremos el siguiente comando:

nvm ls-remote

Procedemos a instalar NodeJS (por fin XD)

nvm install node

Pero si requerimos instalar una versión en específico, podemos despues escribir lo siguiente:

nvm install 14.3 (la versión que queramos)

Para ver las versiones que tenemos instaladas en el sistema:

nvm ls

Para usar cualquiera de esas versiones:

nvm use 14.3 (la versión que necesites)

Casi lo olvidaba... vamos a instalar NPM, que es npm es el sistema de gestión de paquetes por defecto para Node.js, un entorno de ejecución para JavaScript:

sudo apt install npm -y

Para ver la versión de NPM instalada:

npm --version



Instalar Express.js

Express es una aplicación framework de Node.js que provee de un conjunto de características para sitios web y aplicaciones móviles

Por lo que vamos a instalarla con el siguiente comando:

npm install -g express
