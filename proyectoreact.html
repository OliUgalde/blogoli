Crear Proyecto React

Despues de toda esta espera, crearemos por fin nuestro proyecto en React (recuerda hacerlo como super usuario)

Puedes ir a la pagina oficial de React y seguir el tutorial que ahi ofrecen en la siguiente liga:

https://es.reactjs.org/tutorial/tutorial.html

Pero ya que estas aqui... te lo voy a explicar...

Lo primero que vamos a hacer es crear la carpeta donde vamos a alojar nuestra app React, por medio del siguiente codigo:

mkdir (el nombre que tu quieras, sin el paréntesis*)
cd (nombre de tu carpeta)

Vamos a crear el paquete .json para tener node.js

npm init -y
npm install express

Una vez hecho esto, vamos a ingresar la siguiente linea de código para crear la carpeta de my-app (le puedes poner como tu quieras) dentro de la carpeta que tu designaste:

npx create-react-app my-app

Despues de esto, la consola nos va a decir que se ha creado una nueva carpeta llamada my-app (o como le hayas puesto)
Por lo que vamos a meternos a dicha carpeta y eliminar los archivos de la carpeta src con los siguientes comandos:

cd my-app
cd src
rm -f *

Vamos quedarnos en la carpeta src y vamos a crear dos archivos llamados index.css e index.js respectivamente

touch index.css
touch index.js

Ahora tenemos que copiar el código de nuestra app dentro de dichos archivos, pero en este caso, vamos a terminar con el tutorial de la pagina oficial de React y su ejercicio de tic tac toe:

Puedes abrir los archivos con tu editor de código preferido o puedes hacerlo directo en la consola con los siquientes comandos:

sudo nano index.css

Copia el código del siguiente enlace

sudo nano index.js

Copia el código del siguiente enlace
Sin cerrar el archivo JS, vamos a poner el siguiente codigo por encima del que copiamos en el enlace anterior:

import React from 'react';
import ReactDOM from 'react-dom/client';
import './index.css';

Ya que todo está correcto ahora si podemos iniciar npm y nos va a indicar que podemos ir a una url en el navegador para ver nuestra app con el juego tic tac toe con el siguiente comando:

npm start
