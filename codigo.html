React Permission Denied


Intentas crear una app con react con el código:

npx create-react-app

Y a pesar de estar en la consola como superusuario, te arroja el error de:

Puedes ver mi video en youtube que te dejo en seguida explicando paso a paso la solución de este problema y/o leer este blog donde te explico todo...

No desesperes, la solución es mas simple de lo que imaginas... aunque un poco laboriosa, pero empecemos.

Cuando tienes un solo usuario, la cuenta de administrador, ubuntu te arroja ese error como una medida de seguridad, por lo que es necesario crear un nuevo usuario que también pueda logearse como superusuario.

Para ello, vamos a crear un nuevo usuario y vamos a escribir los siguientes comandos:

sudo su
adduser (nombre del nuevo usuario, sin parentesis*)

Desde luego te va a pedir que establezcas una contraseña, anotala para no olvidarla :P

Por último vamos a darles los atributos de administrador al nuevo usuario

adduser (nombre del usuario) sudo

Muy bien!!! ahora el nuevo usuario también tiene atributos de super usuario.

Para pasar de un usuario a otro en la consola ahora solo vamos a escribir:

su (nombredeusuario)

Ahora ya podemos seguir con la instalación de NodeJS...




Instalar NodeJS

Ya que creamos al nuevo usuario, podemos continuar con la correcta instalación de NodeJS

En este blog vamos a instalar NodeJS por medio de NVM (Node Version Manager) para lo que vamos a escribir en la consola: (no olvides que debes estar logeado como super usuario con sudo su)

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

Ya que lo instalamos, vamos a verificar la version de NVM:

nvm -v

Para ver las versiones de node que ahora tenemos disponibles, corremos el siguiente comando:

nvm ls-remote

Procedemos a instalar NodeJS (por fin XD)

nvm install node

Pero si requerimos instalar una versión en específico, podemos despues escribir lo siguiente:

nvm install 14.3 (la versión que queramos)

Para ver las versiones que tenemos instaladas en el sistema:

nvm ls

Para usar cualquiera de esas versiones:

nvm use 14.3 (la versión que necesites)

Casi lo olvidaba... vamos a instalar NPM, que es npm es el sistema de gestión de paquetes por defecto para Node.js, un entorno de ejecución para JavaScript:

sudo apt install npm -y

Para ver la versión de NPM instalada:

nvm --version




Instalar Express.js

Express es una aplicación framework de Node.js que provee de un conjunto de características para sitios web y aplicaciones móviles

Por lo que vamos a instalarla con el siguiente comando:

npm install -g express




Crear Proyecto React

Despues de toda esta espera, crearemos por fin nuestro proyecto en React (recuerda hacerlo como super usuario)

Puedes ir a la pagina oficial de React y seguir el tutorial que ahi ofrecen en la siguiente liga:

https://es.reactjs.org/tutorial/tutorial.html

Pero ya que estas aqui... te lo voy a explicar...

Lo primero que vamos a hacer es crear la carpeta donde vamos a alojar nuestra app React, por medio del siguiente codigo:

mkdir (el nombre que tu quieras, sin el paréntesis*)
cd (nombre de tu carpeta)

Vamos a crear el paquete .json para tener node.js

npm init -y
npm install express

Una vez hecho esto, vamos a ingresar la siguiente linea de código para crear la carpeta de my-app (le puedes poner como tu quieras) dentro de la carpeta que tu designaste:

npx create-react-app my-app

Despues de esto, la consola nos va a decir que se ha creado una nueva carpeta llamada my-app (o como le hayas puesto)
Por lo que vamos a meternos a dicha carpeta y eliminar los archivos de la carpeta src con los siguientes comandos:

cd my-app
cd src
rm -f *

Vamos quedarnos en la carpeta src y vamos a crear dos archivos llamados index.css e index.js respectivamente

touch index.css
touch index.js

Ahora tenemos que copiar el código de nuestra app dentro de dichos archivos, pero en este caso, vamos a terminar con el tutorial de la pagina oficial de React y su ejercicio de tic tac toe:

Puedes abrir los archivos con tu editor de código preferido o puedes hacerlo directo en la consola con los siquientes comandos:

sudo nano index.css

Copia el código del siguiente enlace

sudo nano index.js

Copia el código del siguiente enlace
Sin cerrar el archivo JS, vamos a poner el siguiente codigo por encima del que copiamos en el enlace anterior:

import React from 'react';
import ReactDOM from 'react-dom/client';
import './index.css';

Ya que todo está correcto ahora si podemos iniciar npm y nos va a indicar que podemos ir a una url en el navegador para ver nuestra app con el juego tic tac toe con el siguiente comando:

npm start
